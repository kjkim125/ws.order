주문 데이터 업로드

발주서 형식의 엑셀 파일(.xlsx)을 업로드하면 DB에 저장됩니다. 1행은 머리글로 인식하고 실제 데이터는 2행부터 검사합니다.

엑셀 파일(.xlsx)을 이곳에 드래그하거나 클릭해서 선택하세요

업로드 기록

업로드한 파일명, 등록 건수, 업로드 일시와 로그인 계정을 확인합니다.

업로드 기록을 불러오려면 새로고침을 눌러주세요.

출고일을 선택해 주세요

엑셀에 출고일이 입력되지 않은 데이터가 있습니다. 선택한 날짜를 누락된 행에 자동으로 입력한 뒤 업로드합니다.

관리자 도구

조회 결과 또는 전체 DB에서 필요한 데이터만 안전하게 삭제합니다.

선택 행 삭제는 먼저 주문을 조회하고 왼쪽 체크박스로 행을 선택하면 활성화됩니다. 삭제 전 대상 건수를 다시 확인합니다.
